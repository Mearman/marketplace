# JSON Tools Configuration

Project-specific settings for JSON Tools plugin.

## Settings

- **autoBackup**: Create `.bak` files before edits (default: true)
- **backupLocation**: Directory for backup files (default: ./)
- **defaultIndent**: Default indentation spaces (default: 2)
- **defaultValidation**: Validate after edits (default: false)
